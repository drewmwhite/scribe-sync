    Platform setup notes
    macOS — No setup required. Works out of the box in Chrome or Edge.
    Linux — You may need a udev rule to allow non-root USB access. Run once:
    echo 'SUBSYSTEM=="usb", ATTR{idVendor}=="1949", MODE="0664", GROUP="plugdev"' | sudo tee /etc/udev/rules.d/60-kindle.rules && sudo udevadm control --reload-rules && sudo udevadm trigger
    Then add yourself to the plugdev group: sudo usermod -aG plugdev $USER and log out/in.
    Windows — Windows installs its own MTP driver which blocks WebUSB. You must replace it with WinUSB using Zadig: open Zadig → Options → List All Devices → select your Kindle → choose WinUSB → Replace Driver. Note: after this, your Kindle will no longer appear in File Explorer as a portable device. You can revert by reinstalling the original driver in Device Manager.
